# 🎸 Bocchi The Rock Website

An aesthetic Bocchi The Rock inspired website built with HTML and CSS Grid.

This project was made to practice advanced CSS layouts, Grid, visual hierarchy, section flow, and cinematic website design. Instead of making a basic practice project, I wanted to create something that actually felt emotional, atmospheric, and alive. Especially through the use of background videos, galleries, and animated sections.

Honestly, this project started as “just another Grid practice website” and somehow turned into a full 7-hour coding session until 3 AM 💀

⸻

## ✨ Features

* CSS Grid layouts
* Fullscreen hero section
* Background videos
* Anime-inspired aesthetic
* Image galleries
* Smooth transitions
* Scroll animations
* Hover animations
* Fade-in effects
* Cinematic section reveals
* Animated text and visual effects
* Interactive UI elements
* Responsive section structure
* Smooth visual flow

⸻

## 🛠️ Built With

* HTML5
* CSS3
* CSS Grid
* Flexbox
* Git & GitHub

⸻

### ⚠️ Note

Some animations and transitions were created with AI assistance since I haven’t fully learned advanced animation techniques yet. I still studied, modified, and integrated them into the project myself as part of the learning process.

⸻

## 🎯 Purpose

This project was built to:

* practice CSS Grid deeply
* improve layout design skills
* experiment with aesthetics and atmosphere
* learn how to structure larger frontend projects
* continue improving as a future software engineer
